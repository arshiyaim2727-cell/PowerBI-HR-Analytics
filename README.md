# HR Analytics Dashboard – Power BI

## Project Overview
This Power BI project focuses on analyzing HR data to identify
employee attrition trends and workforce insights.

## Tools & Technologies
- Power BI Desktop
- Excel 
- DAX

## Dashboard Highlights
- Total employees and attrition rate
- Department-wise attrition
- Job role analysis
- Gender and age distribution
- Salary analysis

## hr_data
The hr_data contains employee details such as age, gender,
department, job role, salary, and attrition status.

## Repository Structure
- HR_Analytics_PowerBI.pbix – Power BI report
- hr_data file – Source data
